WEB

Pasos para publicar una página web en la red

1. Elegir un servicio de Hosting

Lo primero que se debe elegir al momento de subir la página web a la red será el proveedor de hosting web, se debe elegir a un buen proveedor de hosting el cual integre las características clave para impulsar el proyecto web. Aspectos a tener en cuenta:

•Soporte en vivo:

Se debe buscar un proveedor de hosting web que brinde soporte, bien sea mediante chat en vivo o en una línea telefónica, esto con el objetivo que, si se presenta un inconveniente, este se resolverá fácilmente.

•Control sobre tu espacio de hosting web:

Se debe tener siempre el mayor control posible sobre la cuenta, así se presentarán menos problemas cuando el sitio web empiece a crecer.

•Espacio para el crecimiento:

Siempre se debe apostar a largo plazo, se debe buscar un proveedor que tenga soluciones escalables en caso de que el sitio web requiera mayor potencia.

•Garantía de devolución de dinero:

Revisar la política de reembolso de el proveedor, esto nos dará tiempo para probar el servicio.

•Beneficios adicionales:

Buscar un buen proveedor de hosting el cual ofrezca ofertas a sus usuarios.

2. Registrar un nombre de dominio

Es la dirección que se le da a la pagina web, mediante la cual puede ser encontrada por quien lo desee.
•Dominio Gratuito: www.empresadehosting.com/usuario/web/index.html
•Dominio Pago:miweb.com

3. Elegir el metodo para subir el sitio web

Es importante elegir la herramienta adecuada para subir la pagina web, existen varias herramientas, pero las más importantes son:

•Administradores de archivos:

Es una herramienta basada en navegador web, tiene todas las características clave para encargarse de los archivos y directorios, pero este presenta un limite de carga, si la copia de seguridad tiene mas de 256 Mb se deben usar otras herramientas.

•Protocolo de transferencia de archivos (FTP):

En esta no existen limites de tamaño, se pueden subir copias de seguridad sin importar el tamaño.

•Importador automático de sitios web:

En este se puede extraer u archivo web de hasta 256 Mb directamente al directorio public_html.

4. Subir el archivo del sitio y extraerlo

Una vez ya realizados lo pasos anteriores subiremos el archivo, daremos clic en el icono de Subir archivos, luego se debe seleccionar el archivo del sitio web en la computadora e importarlo al servidor, por ultimo se usa la función de extraer para descomprimir el archivo. Esto si se realiza por medio de administradores de archivos. Si se usa FTP con FileZilla se debe extraer el archivo desde la computadora, luego se podrán transferir todos los archivos directamente a public_html.

5. Asegurarse de que todos los archivos estén en public_html

Se debe asegurar que todos los archivos estén ubicados en el directorio raíz del dominio, el cual es public_html.

6. Importar la base de datos

Si la pagina web utiliza bases de datos se deben realizar los siguientes pasos:

•Crear una nueva base de datos y usuario MySQL.

•Acceder a la base de datos recién creada a través de phpMyAdmin.

•Utilizar la sección Import para subir el archivo de copia de seguridad.

•Actualizar los detalles de conexión de la base de datos MySQL (database name, host, user, password) en tus archivos de configuración.

7. Comprobar que el sitio web funcione correctamente

Luego de subir todos los archivos del sitio web, se debe revisar el funcionamiento del mismo, entrando al dominio adquirido mediante un navegador web.
